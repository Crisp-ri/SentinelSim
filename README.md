# SentinelSim

**SentinelSim** is a local security analysis platform designed to simulate cyber attacks and detect suspicious activity in real time.

The system combines an attack simulator, detection engine, database storage, and visualization dashboard into a single modular architecture.

---

# Key Features

- Simulation of common attack scenarios

- Real-time detection algorithms

- Suspicious score calculation per IP

- Event storage and alert generation

- Interactive web dashboard

---

# Technology Stack

Backend:

- Python

- FastAPI

- SQLAlchemy

- SQLite

Frontend:

- React

- Vite

- Chart.js / Recharts

---

# Architecture

The system follows a modular pipeline:

Data Source → Detection Engine → Database → Dashboard
